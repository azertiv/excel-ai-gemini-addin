                Features ▼
            

            

                

                    Les fonctions acceptent un paramètre options JSON (par ex. { "lang": "fr", "maxTokens": 1024, "temperature": 0.2 }) pour contrôler la langue, le ton ou le cache.
                


                

                    

                        

                            =AI.ASK
                            Q&A
                        

                        
Pose une question libre avec contexte facultatif pour obtenir une réponse textuelle nette.

                        
    
                            
  • Syntaxe : =AI.ASK(prompt, [contexte], [options])
    • 
                            
  • Utilisation : ajoutez une plage en contexte pour guider la réponse (FAQ, extraits).
    • 
                            
  • Retour : texte court prêt à coller dans une cellule.
    • 
                        

                    


                    

                        

                            =AI.TABLE
                            Données
                        

                        
Génère un tableau structuré en plusieurs lignes/colonnes à partir d'un prompt.

                        
    
                            
  • Syntaxe : =AI.TABLE(prompt, [contexte], [options])
    • 
                            
  • Utilisation : précisez le format attendu (colonnes, unités, volume) dans le prompt.
    • 
                            
  • Retour : matrice prête à être étendue dans la feuille.
    • 
                        

                    


                    

                        

                            =AI.EXTRACT
                            Extraction
                        

                        
Repère et extrait des entités depuis une plage (e-mails, dates, montants, etc.).

                        
    
                            
  • Syntaxe : =AI.EXTRACT(instruction, plage, [options])
    • 
                            
  • Utilisation : indiquez l'entité recherchée (« e-mail pro », « montant TTC », etc.).
    • 
                            
  • Retour : colonne nettoyée avec une valeur par ligne.
    • 
                        

                    


                    

                        

                            =AI.FILL
                            Par exemple
                        

                        
Génère des complétions basées sur vos exemples et les cibles à remplir.

                        
    
                            
  • Syntaxe : =AI.FILL(exemples, cibles, instruction, [options])
    • 
                            
  • Utilisation : fournissez quelques lignes d'exemples alignées sur les cibles.
    • 
                            
  • Retour : liste verticale correspondant aux lignes cibles.
    • 
                        

                    


                    

                        

                            =AI.SUMMARIZE
                            Synthèse
                        

                        
Condense un texte ou une plage en quelques phrases ou puces.

                        
    
                            
  • Syntaxe : =AI.SUMMARIZE(texte, [options])
    • 
                            
  • Utilisation : définissez longueur ou style via options (ex. { "lang": "en" }).
    • 
                            
  • Retour : résumé concis limité aux caractères de cellule.
    • 
                        

                    


                    

                        

                            =AI.TRANSLATE
                            Langues
                        

                        
Traduit un texte vers la langue cible en conservant le sens.

                        
    
                            
  • Syntaxe : =AI.TRANSLATE(texte, langue, [options])
    • 
                            
  • Utilisation : passez un code de langue (« en », « es »...) ou un nom (« français »).
    • 
                            
  • Retour : texte traduit sans guillemets ni balises.
    • 
                        

                    


                    

                        

                            =AI.CLASSIFY
                            Etiquettes
                        

                        
Attribue une étiquette unique parmi une liste fournie.

                        
    
                            
  • Syntaxe : =AI.CLASSIFY(texte, labels, [options])
    • 
                            
  • Utilisation : fournissez les labels sous forme de plage ou de chaîne séparée par « | ».
    • 
                            
  • Retour : une seule étiquette (ou UNKNOWN en cas de doute).
    • 
                        

                    


                    

                        

                            =AI.CLEAN
                            Nettoyage
                        

                        
Nettoie et normalise un texte (espaces, ponctuation, e-mails obfusqués…).

                        
    
                            
  • Syntaxe : =AI.CLEAN(texte, [options])
    • 
                            
  • Utilisation : idéal pour préparer des données avant analyse ou export.
    • 
                            
  • Retour : texte épuré et homogène.
    • 
                        

                    


                    

                        

                            =AI.FORMULA
                            Formules
                        

                        
Génère une formule Excel valide à partir d'une consigne et d'un contexte.

                        
    
                            
  • Syntaxe : =AI.FORMULA(instruction, [contexte], [options])
    • 
                            
  • Utilisation : décrivez le résultat attendu ; ajoutez une plage pour guider la structure.
    • 
                            
  • Retour : formule commençant par « = », prête à être remplie.
    • 
                        

                    


                    

                        

                            =AI.KEYSTATUS
                            Diagnostic
                        

                        
Vérifie rapidement si la clé API est enregistrée côté add-in.

                        
    
                            
  • Syntaxe : =AI.KEYSTATUS()
    • 
                            
  • Utilisation : utile dans un tableau de bord pour suivre la configuration.
    • 
                            
  • Retour : OK ou MISSING.
    • 
                        

                    


                    

                        

                            =AI.TEST
                            Ping
                        

                        
Déclenche un test minimal vers l'API pour vérifier la connectivité.

                        
    
                            
  • Syntaxe : =AI.TEST()
    • 
                            
  • Utilisation : insérez-le dans une cellule pour surveiller la disponibilité.
    • 
                            
  • Retour : OK en cas de succès ou un code d'erreur.
    •
